# Angular-Fengtingxun(Angular扩展)

## 安装

### 一.模块化安装

1. 先下载安装nodejs [https://nodejs.org/en/](https://nodejs.org/en/)

2. 进入nodejs安装目录点击右键bash打开

3. 执行命令

```

#npm安装
$ npm install -g

#安装bower
$ npm install bower -g

#切换到自己的项目目录
$cd yourpath

# 安装fengtingxun
$ bower install fengtingxun

```
